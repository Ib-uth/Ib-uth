## Who I Am

I'm a Senior Full-Stack Engineer and IT Manager based in Abuja, Nigeria.

I've spent the last eight years building things that real people use — intelligence platforms for government clients, fintech dashboards, SaaS products, e-commerce systems, and digital infrastructure for nonprofits. I work across the full stack: React and Next.js on the frontend, Python (FastAPI) and Node.js on the backend, PostgreSQL and MongoDB for data, and Docker plus GitHub Actions for deployment.

My degree is in Cyber Security Science, which means I think about access control, data integrity, and threat surfaces before I write the first line of code. Not as an afterthought.

I'm currently building **SecureFlow** (an API threat intelligence platform) and **NaijaFinance** (a personal finance dashboard built specifically for Nigerian users). Both are open for collaboration.
